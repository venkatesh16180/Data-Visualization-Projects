# HW4 — Arizona Population Demographics Visualization
**Course: INFO 526 | University of Arizona**

## 📌 Overview
Visualization of Arizona county-level population estimates from the 
2023 US Census — exploring demographic trends, population distribution 
across counties, and year-over-year changes using ggplot2.

## 📊 Dataset
`co-est2023-pop-04.xlsx` — US Census Bureau county population 
estimates for Arizona (2020–2023), sourced from the American 
Community Survey.

## 🔑 Key Visualizations
- Bar charts of population by county
- Line plots of population change over time
- Demographic comparison visualizations
- Sorted and labeled county-level plots

## 🛠️ Tools
`R` `ggplot2` `dplyr` `tidyverse` `readxl` `R Markdown`
