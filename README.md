# Data Visualization Projects
**Course: Data Analysis & Visualization (INFO 526) | University of Arizona**

---

## 📌 Overview
A collection of data visualization projects and assignments built in R 
and Python — covering static and interactive visualizations across 
real-world datasets including Olympics athlete data, political PAC 
funding, COVID survey responses, and Arizona population demographics.

---
