# Project 1 — Olympics Dataset Analysis
**Course: INFO 526 | Team: InfoCrew | University of Arizona**

## 📌 Overview
Exploratory data analysis and visualization of the TidyTuesday 2024 
Olympics dataset (271,116 athlete appearances) — investigating medal 
distribution by country and the growth of female Olympian participation 
across decades.

## 🔍 Research Questions
1. Which countries dominate Olympic medal counts and how has this 
   changed over time?
2. How has female athlete participation grown across Olympic games 
   over the decades?

## 🛠️ Tools & Technologies
`R` `ggplot2` `dplyr` `tidyverse` `lubridate` `Quarto` `GitHub`
