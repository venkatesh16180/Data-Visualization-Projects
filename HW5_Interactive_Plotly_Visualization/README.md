# HW5 — Interactive Data Visualization with Plotly
**Course: INFO 526 | University of Arizona**

## 📌 Overview
Building interactive data visualizations using Plotly in R — 
demonstrating the transition from static ggplot2 charts to 
interactive, web-ready visualizations with hover effects, 
zoom controls, and dynamic filtering.

## 🔑 Key Visualizations
- Interactive scatter plots with hover tooltips
- Interactive bar charts with click filtering
- Animated time series visualizations
- Comparison of static ggplot2 vs interactive Plotly output

## 🛠️ Tools
`R` `Plotly` `ggplot2` `dplyr` `R Markdown`
